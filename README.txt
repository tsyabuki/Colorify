README:

Installation:
-Run the Colorify.py file once. An error will occur, but the template ini file will be created.

Running the program:
-Fill out the ini file with relevant information to run the program for color changinng the image file
-Ensure that the output file type is a PNG or JPG format
-Ensure the input file is within the same directory as Colorify.py
-The Overwrite function will completely overwrite the original file REGARDLESS of the output file name
-Run the Colorify.py file again
-The output file should be created or overwriten into the same directory as the Colorify.py file
